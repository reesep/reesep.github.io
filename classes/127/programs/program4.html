Background Information

  • Scorigami is the art of building final scores that have never happened before in NFL history. Due to the unique nature of how points are scored in (American) Football, where it is impossible to score 1 point on its own, as well as the rarity of the 2 point safety and 8 point touchdown and 2 point conversion, there are a lot of scores that are possible, but have never happened.
    • For example, on December 26th, 2020, the final score between the Miami Dolphins and Las Vegas Raiders was 26-25. This game ended in a Scorigami. This was the first time in NFL history where a game ended in the score of 25-26. On Christmas day, there was also a final score of 33-52 between the Minnesota Vikings and New Orleans Saints. This was also scorigami. To learn more about scorigami, watch this super cool video. You can also visit this website to visualize all the different scorigamis that have happened in the past 50+ years For this assingment, you will be building a program that reads in information from a file of NFL scores since 2019. You will be using a dictionary to hold information about different NFL scores.

Assignment

  • Using program4.py (renamed according to the instructions above) as a starting point, supply the missing functions, missing class, and missing class methods. You will need to implement six different functions:
  • fill_scorigami_dictionary() should read in from nfl_scores.csv and fill a dictionary with information with all unique scores. At the end of this function, you should return the filled dictionary
    • The keys for the dictionary should be a tuple that contains a certain score. (21,7), (30,15), etc. NOTE: the score (X,Y) is the same score as (Y,X) and should be treated as the same score
    • The values for the dictionary should be a Scorigami object . Each scorigami object has a (1) Home Team (2) Away Team (3) Week (4) Year and (5) Location/Stadium. You will need to extract this information from the file.
    • This means you will define a Scorigami class. This class will need a constructor and __str__ method. You will also include any other reader/writer methods that you need. (getYear(), getHomeTeam(), etc)
  • For all remaining functions, you should NOT open the file anymore . You should only be using your dictionary with information
  • The function get_information_for_score should allow the user to search for a score in the dictionary. If the score exists, it should print out what two teams were involved, the week, the year, and location of the game. If the score has not happened yet, then "That score has not happened yet!" should be printed out. See sample output
  • The function search_scorigamis_by_team should allow the user to search for all Scorigamis that involved a certain team. The function should print out information for each game as well as the total number of Scorigamis found. For example, if the user searches for "Oakland Raiders", then it should print out the total number of Scorigamis that included the Oakland raiders and information for all Scorigamis where the Oakland Raiders were either the home team or away team. Note that the user should be able to put in a team name in all lowercase, uppercase, or a mixture of both. See sample output
  • The function search_scorigamis_by_year should allow to user to input a year. The function should then print out the number of Scorigamis that happened that year and information for each of the Scorigami games that occured.
  • The function team_with_most_scorigamis should calculate which team was involved in the most Scorigamis and how many Scorigamis they were involved in. The answer should always be "San Francisco 49ers with 99 total scorigamis". Hint: consider creating a new dictionary, where the keys will be each team, and the values will be the total number of scorigamis involving that team. You might find the billionaires.py example helpful here...
  • The function print_menu() is a very simple function that should print out the menu that you see in the sample output
