Example Test Questions

  • Suppose you have a dictionary containing cities and their respective populations
    • 		places = {  
		"Billings, MT": 109550,
		"Seattle, WA": 744995,
		"Layton, UT": 77303,
		"Bozeman, MT": 48532,
		"Missoula, MT": 74428,
		"Omaha, NE": 468262,
		"Polson, MT": 5018
		}
    Write a function that takes in this dictionary of information and sums up the populations of only Montana cities (Billings, Bozeman, Missoula, Polson).

  • Suppose you have a dictionary containing coordinates (latitude, longitude) and the city located at those coordinates 
    • 		places = {
		(23.1136,-82.3666):"Havana, Cuba",
		(14.6349, -90.5069):"Guatamala City, Guatamala",
		(32.7767, -96.7970): "Dallas, Texas",
		(13.7563, 100.5018): "Bangkok, Thailand",
		(-22.9068,-43.1729): "Rio de Janiro, Brazil",
		(-12.0464, -77.0428): "Lima, Peru",
		(52.5200, 13.4050): "Berlin, Germany",
		(14.7167, -17.4677): "Dakar, Denegal"
		}
    Write a function that takes in this information and counts how many cities in the dictionary are located between the equator (latitude of 0) and the tropic of cancer (latitude 23.5). In the example dictionary provided, the answer should be 4.
