Midterm Exam Overview

The following content is fair game on the Midterm Exam
  • Data Types/Variables
  • Operators (+, -, *, %, ...)
  • User Input
  • .format
  • Functions
  • Selection
  • Iteration (for/while loops)
  • Lists
  • Nested Lists
  • Strings
  • Modules (math/random)
  • Files (reading/writing)

Midterm Exam Format

100 Points total
  • 10 Multiple choice / True or False questions (2 pts each)
      • Questions such as:
    • What does this code print out?
    • What is wrong with this code?
    • Which of the following is/is not _____
    • Other typical MC questions about concepts we discussed in class
      • I will pull two of these questions directly from the practice exam.
  • 3 short programming problems
    • Given a problem, you will write a Python solution that will solve it. You will need to write a function or a program (no function required) to solve the problem
    • Problem 1: Basic Python Program / Computation (20 pts)
    • Problem 2: Selection / If Statements (25 pts)
    • Problem 3: Nested Lists (35 pts)
    I will pull one of these problems directly from one of the example test questions from the course website. These questions are similar in size to most of the examples we've done in class.

Tips

Be sure to attempt every problem. Even if you cannot fully solve one the programming problems. You should try to get as much done as you can. I will give out as much partial credit as I can even if you cannot fully solve it.
Read the multiple choice problems and the given code carefully. Some questions will be intentionally tricky.
Read the programming problems instructions carefully. Make sure you understand what is being asked and what you need to do (write a function, what the inputs are, what the output should be, etc)
