Background and Directions

It's often difficult to figure out where to eat at in Bozeman. You will be creating a program that asks the user for (1) what kind of food they are interested in and (2) how much money the have. The user will only answer with the following kinds of food: Asian, Mexican, Italian, Pizza, or Burgers. The user can have any amount of money Using that information, you will use logic to recommend a restaurant to the user. The logic, which you will write, will be in a function called determine_restaurant(). This function will return the name of the restaurant that it recommends. The logic used in your program should follow the following criteria:

  • If the user ever has less 5 dollars or less, then "Nowhere! You need more money!" should be returned
  • If the user selects Asian food and has between 5 and 12 dollars, then "Panda Express" should be returned
  • If the user selects Asian food and has 12 or more dollars, then "Dave's Sushi" should be returned
  • If the user selects Mexican food and has between 5 and 12 dollars, then "Taco Bell" should be returned
  • If the user selects Mexican food and has 12 or more dollars, then "Sante Fe Reds" should be returned
  • If the user selects Italian food and has between 5 and 12 dollars, then "Olive Garden" should be returned
  • If the user selects Italian food and has 12 or more dollars, then "Blacksmith Kitchen" should be returned
  • If the user selects pizza and has between 5 and 12 dollars, then "Domino's Pizza" should be returned
  • If the user selects pizza and has 12 or more dollars, then "Bridger Brewing should be returned
  • If the user selects burgers and has between 5 and 12 dollars, then "McDonalds" should be returned
  • If the user selects burgers and has 12 or more dollars, then "Burger Bob's" should be returned

Using lab3.py as a starting point , define and fill in the determine_restaurant() function so that the correct restaurant recommendation is returned. You might find this flowchart helpful for understanding the logic of the program
