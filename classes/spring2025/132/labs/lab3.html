Background and Directions

In this assignment, you will be forming a dictionary in your program from reading from a file. You will use words.txt. To fill your dictionary. This file has most of the words in the english dictionary.

Using Lab3Demo.Java, as a starting point, you will supply the necessary classes and methods to get the correct output. You can download this file, or copy and paste it into your own Lab3Demo.java.

You first need to define the instance fields and constructor for the Dictionary class. There will be at least one instance field. You will need to keep track of all the words from the input file, which will be a String[]. You will only read from the file once. I would recommend writing a method, perhaps called readFromFile() which will open the file, and then return the filled array of Strings.

public Dictionary() {

   this.words = readFromFile();

}

There are 41193 words in the input file. You can hardcode this value in your code when creating the array. The words are not in alphabetical order. You will need to make sure the words are sorted alphabetically in your dictionary before moving to the next part.

In the Dictionary class, you will need to define the following methods:

  • lastWord()- returns the last word of the dictionary.
  • longestWord()- computes and returns the longest word in the dictionary. Hint: there is a .length() method that will return the length of a string.
  • wordsWithAllVowels() - computes and returns the number of words in the dictionary that contain every vowel (a, e, i, o, u) at least once. For example, "neurologically" and "unforgivable" are examples of words that contain every vowel at least once.
  • medianWordLength() - computes and returns the median words length of all the words from the dictionary. Given a sorted array of the word lengths of each word, the median is the middle value. If you want to treat the median as a double (8.0) instead of an int, that is fine.
  • countByLetter(char inputLetter)- given an input letter (a char), this will count and return the number of words in the dictionary that start with that letter. Hint: there is a .charAt(X) that will return the character at index X of a string. You can assume the use will always enter a valid, lowercase letter.
