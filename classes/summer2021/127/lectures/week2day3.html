Example Test Questions

  • Write a program that calculates the average word length in a sentence entered by a user

  • Write a python function that reverses a list of items WITHOUT using the built in .reverse() function

  • Write a function replace(s, old, new) that replaces all occurences of old with new in a string s Hint: use the .split() and .join() functions. For example, when replace("Hello world","Hello","Hola") is called, then "Hola World" will be printed out.

  • Consider a list where the words "start" and "end" will always appear at some index, however you can assume that "start" will always appear before "end". For example ["start","hello","dog","cat","end","foo"] or ["foo","bar","start","dog","end"]. Your job is to write a python function that will take in a list of any size. The function should return the slice of the list that starts with "start" and ends with "end". For example, if ["foo","bar","start","dog","end"] is the input, then ["start","dog","end"] should be returned. If ["start","hello","dog","cat","end","foo"] is the input, then ["start","hello","dog","cat","end"] should be returned.
