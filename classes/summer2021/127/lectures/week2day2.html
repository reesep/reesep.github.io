Example Test Questions

  • Write a function that prints out all multiples that are under 1000 for some input value of n. For example, if n=127, then 127, 254, 381, 508, 635, 762, and 889 should be printed out.

  • Write a program to ask for a name until the user enters "END". Print the name each time. When you are done, the program should print "I am done."

  • Convert the following for loop into an equivalent while loop: 
    • 		for i in range(1,10):
			print(i)

	3 3 6 9 12 15
