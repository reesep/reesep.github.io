  • Write a program that will calculate calculate the area of a triangle. The final answer should only include 4 decimal places.

  • Write a python function that will take in the cost of a dinner bill at a restaurant. The function should then prompt the user for what percent tip they want to leave (20%, 15%, etc). The function should calculate the amount to tip the waiter/waitress based on what percent tip they want and the order total. For example, if the dinner bill is $100 and they want to leave a %20 tip, the function should return $20 for a tip.

  • Write a python program that will ask the user for a number of cents (5¢, 99¢, 23¢, etc). The program should print out how to create that number of cents using ONLY pennies (1¢) and dimes (10¢). Your program should minimize the number of coins used. For example, if the user inputs 99¢, your program should print out 9 dimes and 9 pennies. If the user inputs 20¢, your program should print out 2 dimes and 0 pennies
