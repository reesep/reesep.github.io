Example Test Questions

  • Consider a dictionary that contains information abouts golfers and their score for the day
    •   
		golfers = { 
		"Rory McIlroy": 0,
		"Phil Mickelson":-1,
		"Tiger Woods": -5,
		"Jerry Pearsall": 2,
		"Dustin Johnson": -3
		}
    Write a function that will take in this information and determine who the winner is. In golf, the winner is the player with the lowest score


  • Consider a dictionary that contains enrollment information about computer science courses. The values of the dictionary contain information about the instructor, location, and number of students enrolled. 
    • 		classes = { 
		"CSCI 127": ["Reese Pearsall", "Barnard 103", 32],
		"CSCI 107": ["John Paxton", "Leon Johnson 339", 71],
		"CSCI 460": ["Travis Peters", "Barnard 108", 65],
		"CSCI 232": ["Brendan Mumey", "Reid 101", 95],
		"CSCI 347": ["Veronika Neely", "Norm Asbjornson 201", 80]
		"CSCI 538": ["Binhai Zhu", "Barnard 323", 14]
		"CSCI 112": ["Mary Ann Cummings", "Barnard 103", 55]
		"CSCI 366": ["Carson Gross", "Reid 202", 107]
		"CSCI 132": ["Daniel DeFrance", "Norm Asbjornson 165", 121]
		}
    Write a function that will take in this information and sum up the number of students enrolled for all upper division classes (300+ level classes). Hint: The class number will always be the last 3 digits of the string.


  • Using this file that contains information about Fall 2020 CS clases . Write a function that will read this file and use a dictionary to determine which how many students each professor had

  • Suppose you have a dictionary containing cities and their respective populations
    • 		places = {  
		"Billings, MT": 109550,
		"Seattle, WA": 744995,
		"Layton, UT": 77303,
		"Bozeman, MT": 48532,
		"Missoula, MT": 74428,
		"Omaha, NE": 468262,
		"Polson, MT": 5018
		}
    Write a function that takes in this dictionary of information and sums up the populations of only Montana cities (Billings, Bozeman, Missoula, Polson).

  • Suppose you have a dictionary containing coordinates (latitude, longitude) and the city located at those coordinates 
    • 		places = {
		(23.1136,-82.3666):"Havana, Cuba",
		(14.6349, -90.5069):"Guatamala City, Guatamala",
		(32.7767, -96.7970): "Dallas, Texas",
		(13.7563, 100.5018): "Bangkok, Thailand",
		(-22.9068,-43.1729): "Rio de Janiro, Brazil",
		(-12.0464, -77.0428): "Lima, Peru",
		(52.5200, 13.4050): "Berlin, Germany",
		(14.7167, -17.4677): "Dakar, Denegal"
		}
    Write a function that takes in this information and counts how many cities in the dictionary are located between the equator (latitude of 0) and the tropic of cancer (latitude 23.5). In the example dictionary provided, the answer should be 4.
