In order to complete this lab, you will need to understand

  • Matplotlib
  • Pandas (optional)
  • numpy (optional)

Assignment Directions

  • For this assignment, you will be producing two different visualizations that help the user gain insight into the information contained in a csv file
  • You may select any of the csv files from the CORGIS CSV Datasets (although, if you use the weather or earthquakes one, I will call you lame). Using that CSV file, you will create two different types of visualizations (you cannot have two of the same kind of graph)
  • It should be clear to the user what information is being graphed, so you need to include appropriate titles, labels, etc
  • You can use Matplotlib or Pandas (or both) to create the visualizations. Numpy may also be used
  • There should be no user input. Your program should just produce the visualizations
  • You will develop the solution from scratch. I would recommend placing both of your graphs inside into a seperate function of some kind (i.e. def create_graph_one() and def create_graph_two() ).
  • Some CSV files may be easier to work with than others. I've seen this assignment work for the video_games.csv file and airlines.csv file.
