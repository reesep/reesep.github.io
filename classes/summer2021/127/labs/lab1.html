Background and Directions

You will write a program that will a generate a very generic social media profile. Your program should prompt for a (1) first name, (2) last name, (3) birthday, (4) job, (5) favorite movie, and (6) favorite book. The profile will also include a small ASCII art icon as a "profile picture"

You will need to create your own ASCII art icon or find one on the internet and put it in a box, just like this example boat ASCII art:
