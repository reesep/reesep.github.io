Background and Directions

You will be writing a program that will prompt the user a series of math multiplication questions. The program will keep track of how many guesses it takes in order to complete all the problems. The user must successfully answer each question correctly before moving onto the next question. Once all problems are solved, the program will calculate their accuracy (# of correct guesses / # of total guesses). The multiplication problems should include two random numbers between 1 and 10.

Using lab3.py as a starting point, complete the missing function so that the program will simulate the math game described above. Other than the random numbers, your program should look almost exactly the same as the sample outputs below.

Hints

  • Generating random numbers is already done for you. Call the get_random_number() function to get a random number between 1 and 10
  • This lab will require both the use of a for loop and while loop
