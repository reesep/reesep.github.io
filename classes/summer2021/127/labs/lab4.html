Background and Directions

You are NOT allowed to import any external modules (numpy, statistics, math, etc) for this lab. Doing so will result in a major loss of points

You will be writing a program that will calculate the mean, median, and mode of some integer dataset

Using lab4.py as a starting point , define and fill in the missing functions.

  • calculate_mean() should take in the list, calculate the mean of the dataset and print it out in as seen in this screenshot . The mean is the average of all the numbers in the list
  • calculate_median() should take in the list, calculate the median of the dataset and print it out in as seen in this screenshot . The median is the middle value of the sorted integer list. If there is no definite middle value, it will be the average between the two middle values.
  • calculate_mode() should take in the list, calculate the mode of the dataset and print it out in as seen in this screenshot . The mode is the number(s) that occur the most in the list. There may be more than one mode value. You can assume that there will always be at least one mode value.
    • Note: If your program can only identify one of the mode values, you will still receive full credit.

Hints

  • There are lots of helpful examples that we did in lecture. Check them out below
  • Calculating the mean should be pretty easy. For calculating the median, the formula will change depending on whether the size of the list is even or odd. Consider checking if the size of the list is even or odd first.
  • For calculating the mode, consider using the .count() list function discussed in lecture. If you want to go the extra mile and have your program detect all mode values, consider the strategy below:
    • Create an empty list. This is the list that will contain all mode values
    • Whenever you find a new highest mode value, empty the list and add the new value
    • If you find a mode value that is equal to what is already in the mode value list, and the number is not already in the list, add it to the list
    • After your for loop finished, you should have a list that contains your mode values
